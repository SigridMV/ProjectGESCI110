# Micro-gardens Informational Website

This project is an informational website designed to help beginners learn how to grow micro-gardens in small spaces such as apartments, urban homes, or shared living environments. The site includes practical guides, research-based recommendations, beginner-friendly tutorials, and a simple 4-week plan to help anyone start growing fresh herbs and greens at home.

## Features
- Clear explanation of the urban fresh-food access problem  
- Causes, locations, and context  
- Two research questions with summarized findings and credible sources  
- Three practical how-to guides (basil, lettuce, microgreens)  
- Supply lists, cost estimates, and simple materials  
- A 4-week actionable plan for beginners   
- Accessibility-focused, mobile-responsive layout  
- Easy to deploy on Render, Netlify, or Vercel
